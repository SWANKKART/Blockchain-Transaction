HashVault — Blockchain Transactions

// secure peer-to-peer ledger demo · python · sha3-256 · final year project

Python 3.x Google Colab SHA3-256 Blockchain Final Year Project

Abstract

HashVault demonstrates how blockchain technology secures peer-to-peer financial transactions between friends — without requiring a central authority like a bank. Each transaction is cryptographically hashed and chained to the previous block, making the ledger tamper-evident and transparent.

Key capabilities

Cryptographic hashingSHA3-256 generates a unique 256-bit fingerprint for every block of data.
Chain immutabilityAltering any block invalidates all subsequent hashes, detecting tampering instantly.
TransparencyEvery transaction is visible across all nodes in the distributed ledger.
DecentralizationNo single point of failure or authority controls the transaction record.

Core code snippet

# Import built-in hashing library
import hashlib

# Hash a transaction string with SHA3-256
m = hashlib.sha3_256()
m.update(b'Alice sends Bob 500 INR')
tx_hash = m.hexdigest()

print(f'Hash: {tx_hash}')
